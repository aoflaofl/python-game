# python-game

A video game written in Python using [Pygame](https://www.pygame.org/docs/).

This simple game displays a red circle on a purple background. You can move the circle around the screen using the W, A, S, and D keys.

## Getting started

1. Install the dependency:
   ```bash
   pip install pygame
2. Run the game:
   ```bash
   python game.py
   ```
You should see a window with a purple background and a red circle that you can move with the WASD keys.
